How to release

* Update version in code
* Update latest version in markdown docs/version.md
* update docs/ChagneLog.md
* Generate docs, Update usage (make dev;make clean;make)
* Look at README.md and ChangeLog.md
* Compile
* Test
* Update release_notes.md
* commit
* git push

make sure go-xbuild-go is atlease v1.0.5

  export GITHUB_TOKEN=XXX
  go-xbuild-go -release

* Update muquit.com if needed

How to get github access token
==============================
* Profile -> Settings > Developer Settings -> Personal access token -> Token (classic)
